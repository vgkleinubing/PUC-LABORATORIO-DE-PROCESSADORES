Programas

  • a02_pisca_led Primeiro projeto: Pisca LEDS.
    • Makefile do primeiro projeto
    • crt.S Rotina de startup
    • lpc2378_flash.ld Configuração do Linker
    • main.c Programa principal do primeiro projeto

  • a03_uart/ Segundo projeto: Configura a UART e escreve uma mensagem.
    • Makefile do segundo projeto
    • crt.S Rotina de startup
    • lpc2378_flash.ld Configuração do Linker
    • main.c Programa principal do segundo projeto

  • a04_timer_lcd/ Terceiro projeto: Configura Timer e Display LCD.
    • Makefile
    • crt.S Rotina de startup
    • lpc2378_flash.ld Configuração do Linker
    • main.c Programa principal do terceiro projeto

  • a05_relogio/ Quinto projeto: Relógio RTC.
    • Makefile
    • crt.S Rotina de startup
    • lpc2378_flash.ld Configuração do Linker
    • main.c Programa principal do quarto projeto
    • lcd.c Módulo com rotinas para acessar o display LCD
    • lcd.h Cabeçalho do módulo LCD

  • a06_pll/ Sexto projeto: Configuração do Clock e do PLL.
    • main.c Programa principal com um diálogo através da porta serial
    • pll.c Configuração do Clock e do PLL
    • uart.c Configuração da porta serial UART

  • a07_T0_FIQTimer com interrupção vetorada tipo FIQ. Usa a interrumpção do timer T0 para piscar LEDs
    • main.c Programa principal
    • vic_cpsr.h Cabeçalho com macros para habilitar interrupções no microcontrolador.

  • a07_RTC_IRQ Usa uma interrupção do relógio RTC em modo IRQ e para acionar o display LCD. No acionamento do display implemeta-se uma fila tipo FIFO. As escritas destinadas ao display vão para uma fila na memória. A rotina de interrupção retira desta fila um caractere por vez e escreve no LCD.
  • a08_tmp101/ Demonstra o uso da interface I2C sem interrupção. Acessa um termômetro tipo TMP101 conectado à interface I2C numero 2, implementando um relógio com termômetro.
  • a08_tmp_irq/ Similar ao a08_tmp101: (Relógio com termômetro TMP100) porém usando a interrupção da interface I2C.
  • a08_smcard/ Le e escreve em cartões tipo SmartCard, usando a interface I2C.
  • pwm/ Programa que demonstra a configuração do modulador por largura de pulso (PWM).
  • mon23/ Programa que monitor que mostra/edita a memória, carrega e executa prohramas *.hex na RAM, e programa a memória FLASH isando funções IAP.
  • mprintf/ Programa que demonstra o uso de aritmética em ponto flutuante tipo double. Incui uma implementação própria da função sprintf e sqrt.

Bibliografia

Todos os ítens abaixo estão integralmente disponíveis na internet:
  • user.manual.lpc23xx.pdf [12MBytes] Este é o manual completo oficial da NXP sobre a família LPC2368/LPC2378, incluindo a descrição dos periféricos e registradores especiais
  • Trevor Martin: The Insider's Guide To The LPC21xx Livro completo sobre a família ARM da NXP.
  • http://openhardware.net/Embedded_ARM/Toolchain Aqui explica como compilar o seu próprio sistema de desenvolvimento para ARM baseado nas versões mais recentes das ferramentas GNU-GCC.
  • http://www.mikrocontroller.net/articles/LPC2000 Página em Alemão com muitas informações links artigos e programas para download relacionados com os processadores ARM da NXP.
  • http://www.mikrocontroller.net/en/arm-gcc Aqui está disponível para download o conjunto de ferramentas gnu-arm-gcc compilado para Linux e para McIntosh.
  • Martin Thomas: ARM Projects Página com programas exemplo, links e informações sobre processadores LPC21xx. Aqui pode-se obter a versão compilada para Windows do arm-gcc.
  • http://www.nxp.com/pip/LPC2378_3.html Página de apresentação do LPC2378 na NXP.
  • http://www.eclipse.org/cdt/ Este é um ambiente integrado (IDE) para ARM escrito em Java usando as ferramentas GNU. Pode funcionar em Windows ou em Linux.
  • eclipse_ARM.pdf [3,5M copia local] Apostila que descreve como instalar e usar o ambiente integrado eclipse junto com os compiladores GNU para ARM. Escrita por Gustavo Schwartz, a partir de pesquisas na internet. Usou o livro ARM Cross Development with Eclipse de James P. Lynch como principal fonte de informações. Pode-se entrar em contato com o autor em gs.br[arrouba]hotmail.com.
  • http://tech.groups.yahoo.com/group/lpc21isp/ Grupo de discissão no yahoo sobre o lpc21isp
  • http://tech.groups.yahoo.com/group/lpc2000/ Grupo de discissão no yahoo sobre os processadores da família LPC2000
  • http://embeddedrelated.com/groups/lpc2000/1.php Forum de discussão sobre microcontroladores da família LPC2000. Muitas informações úteis.
